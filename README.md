# Personal Skills

Repositório para armazenar **personal skills** que trabalham em conjunto com o projeto [superpowers](https://github.com/obra/superpowers).  
Cada skill aqui complementa o conjunto principal, permitindo estender comportamentos sem alterar o core.

## Objetivo

Fornecer um espaço separado para skills adicionais, organizadas de forma simples e reutilizável.

## Estrutura

- Cada skill fica em sua própria pasta.  
- Código e metadados ficam juntos para facilitar manutenção.  
- Sem dependência direta entre skills. Cada uma funciona de forma isolada.

## Como usar

1. Clone este repositório.  
2. Adicione suas skills seguindo o padrão existente.  
3. Importe no projeto principal conforme a convenção definida no *superpowers*.

## Contribuição

- Pull requests são bem-vindos.  
- Mantenha o formato das skills consistente.  
- Evite acoplamento com outras skills.
